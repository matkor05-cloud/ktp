# Лабораторная работа №3

## Описание

**Задание 1.** Создан обобщённый класс `HashTable<K, V>`, реализующий хэш-таблицу методом цепочек. Внутренний класс `Entry` хранит пару ключ-значение. Реализованы методы:
- `put()` — добавление пары, обновление значения для существующего ключа
- `get()` — получение значения по ключу
- `remove()` — удаление пары по ключу
- `size()` и `isEmpty()`

Хэш-функция использует `hashCode()` ключа и остаток от деления на длину массива.

**Задание 2.** Создан класс `Student` с полями `name`, `surname`, `age`, `averageGrade`. В `Main` используется встроенный `HashMap<String, Student>` для хранения студентов по номеру зачётки. Реализованы операции вставки, поиска и удаления.

## Файлы
- `HashTable.java`
- `Student.java`
- `Main.java`

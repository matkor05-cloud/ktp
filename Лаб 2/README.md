# Лабораторная работа №2

## Описание

Создана иерархия классов по теме «Бытовая техника».

**Абстрактный класс Appliance** содержит поля `brand`, `model`, `powerWatts`, статический счётчик `applianceCount`, конструкторы, геттеры и сеттеры, абстрактные методы `turnOn()` и `performWork()`.

**Класс Refrigerator** (холодильник) наследует `Appliance`, добавляет поле `temperature`, переопределяет абстрактные методы. Демонстрирует перегрузку метода `setTemperature()`.

**Класс VacuumCleaner** (пылесос) наследует `Appliance`, добавляет поле `bagCapacity`, переопределяет методы.

В **классе Main** создаются объекты, демонстрируется полиморфизм (массив `Appliance[]`), ввод данных пользователем и работа статического счётчика.

## Файлы
- `Appliance.java`
- `Refrigerator.java`
- `VacuumCleaner.java`
- `Main.java`

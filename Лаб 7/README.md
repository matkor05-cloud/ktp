# Лабораторная работа №7

## Описание

**Задание 1.** Программа `ArraySum` вычисляет сумму элементов массива в двух потоках. Каждый поток обрабатывает свою половину массива. После завершения потоков главный поток складывает результаты. Используется `Thread` и `join()`.

**Задание 2.** Программа `MatrixMaxFinder` находит наибольший элемент в матрице. Для каждой строки создаётся отдельный поток, который находит максимум в строке. После завершения всех потоков главный поток сравнивает полученные максимумы и выводит глобальный максимум.

**Задание 3.** Программа `WarehouseTransfer` моделирует работу склада с 3 грузчиками. Каждый грузчик — отдельный поток (наследует `Thread`). Грузчики берут товары со склада, пока суммарный вес набранных товаров не достигнет 150 кг, после чего «переносят» их на другой склад. Синхронизация доступа к складу реализована через `synchronized`.

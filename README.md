## Краткое содержание

### Лабораторная работа №1
Программы для поиска простых чисел и проверки строк на палиндром.

### Лабораторная работа №2
Иерархия классов «Бытовая техника» с демонстрацией принципов ООП.

### Лабораторная работа №3
Реализация хэш-таблицы методом цепочек и работа с HashMap для хранения студентов.

### Лабораторная работа №4
Обработка исключений при вычислениях, копировании файлов и создание пользовательского исключения с логированием.

### Лабораторная работа №5
Использование регулярных выражений для поиска чисел, валидации паролей и IP-адресов, обработки текста.

### Лабораторная работа №6
Работа с коллекциями: подсчёт частоты слов, обобщённый стек, учёт продаж.

### Лабораторная работа №7
Многопоточное программирование: сумма массива в двух потоках, поиск максимума в матрице, перенос товаров грузчиками.

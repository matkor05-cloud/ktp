# Лабораторная работа №6

## Описание

**Задание 1.** Программа `TopWords` считывает текстовый файл и выводит топ-10 самых часто встречающихся слов. Используется `HashMap<String, Integer>` для подсчёта частоты. После чтения записи сортируются по убыванию с помощью `Collections.sort()` и компаратора.

**Задание 2.** Создан обобщённый класс `Stack<T>`, реализующий стек на основе массива. Методы:
- `push()` — добавление элемента на вершину
- `pop()` — удаление и возврат верхнего элемента
- `peek()` — просмотр верхнего элемента без удаления

**Задание 3.** Программа `SalesAccounting` для учёта продаж в магазине. Используется `TreeSet` для хранения проданных товаров (автоматическая сортировка по названию). Реализованы: добавление товара, вывод списка, подсчёт общей суммы, определение наиболее популярного товара (с помощью `HashMap` для подсчёта количества продаж).

# Лабораторная работа №1

## Тема
Введение в программирование на Java

## Задание 1: Простые числа
Программа находит и выводит все простые числа меньше 100.

**Алгоритм работы:**
- Метод `isPrime(int n)` проверяет, является ли число простым
- В цикле от 2 до 100 вызывается метод `isPrime()`
- Если число простое — выводится на экран

**Файлы:**
- `Primes.java` — основная программа

## Задание 2: Палиндром
Программа проверяет, является ли введённая строка палиндромом (читается одинаково слева направо и справа налево).

**Алгоритм работы:**
- Метод `reverseString(String s)` разворачивает строку в обратном порядке
- Метод `isPalindrome(String s)` сравнивает исходную строку с перевёрнутой
- В методе `main` обрабатываются аргументы командной строки

**Файлы:**
- `Palindrome.java` — основная программа
